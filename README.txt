# Desarrollado en Apache Haus 2.4, con certificado SSL y URL modificada mediante configuración local.
# Esto significa que, al haber compartido el proyecto fuera de mi PC, ni la URL personalizada ni el certificado SSL se pueden apreciar,
# lo cual me jode, porque tardé horas en conseguir hacer ambas cosas, pero es lo que hay.
#
#
# Para abrir el proyecto y evitar problemas de compatibilidad, se recomienda abrir una consola de comandos (CMD) en el directorio raíz del 
# proyecto y ejecutar el siguiente comando:
#
#
# 	php -S localhost:8000
#
#
# Este comando habilita un servidor local en cualquier ordenador con PHP 7 instalado (lo cual se da por sentado tratándose de desarrolladores web).
#
# 
# Otra opción sería copiar el proyecto dentro de 'htdocs' y ejecutarlo como cualquier otro proyecto hecho en XAMPP o Apache. Sin embargo, esto puede
# llevar a errores si no se tiene bien configurado el servidor local.
